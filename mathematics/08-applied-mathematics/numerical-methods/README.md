# Numerical Methods

Computational algorithms for mathematical problems.

## Overview

Numerical methods provide algorithms to approximate solutions to mathematical problems that may not have closed-form solutions.

## Core Topics

### Root Finding
- Bisection method
- Newton-Raphson method
- Secant method
- Fixed-point iteration

### Interpolation
- Polynomial interpolation
- Lagrange interpolation
- Splines
- Least squares fitting

### Numerical Integration
- Riemann sums
- Trapezoidal rule
- Simpson's rule
- Gaussian quadrature
- Monte Carlo integration

### Numerical Linear Algebra
- LU decomposition
- Gaussian elimination
- Iterative methods
- Eigenvalue algorithms
- SVD computation

### Numerical ODEs
- Euler's method
- Runge-Kutta methods
- Multistep methods
- Stiff equations

### Numerical PDEs
- Finite difference methods
- Finite element methods
- Spectral methods

## Error Analysis

- Stability
- Convergence
