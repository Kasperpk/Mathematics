# Applied Mathematics

Mathematical methods for solving real-world problems.

## Overview

Applied mathematics develops and uses mathematical tools to solve problems in science, engineering, and other fields.

## Sections

### [Numerical Methods](./numerical-methods/)
Computational algorithms for mathematical problems.
- Root finding
- Numerical integration
- Linear algebra algorithms
- Numerical ODEs and PDEs

### [Optimization](./optimization/)
Finding the best solutions.
- Linear programming
- Nonlinear optimization
- Convex optimization
- Discrete optimization

### [Differential Equations](./differential-equations/)
Modeling dynamic systems.
- Ordinary differential equations
- Partial differential equations
- Dynamical systems
- Chaos theory

## Learning Path

1. Build strong foundations in calculus and linear algebra
2. Study differential equations for modeling
3. Learn numerical methods for computation
4. Apply optimization for decision-making

- Physics simulations
- Engineering design
- Financial modeling
