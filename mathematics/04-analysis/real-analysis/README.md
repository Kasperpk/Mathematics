# Real Analysis

The rigorous foundation of calculus.

## Overview

Real analysis provides the theoretical framework for understanding limits, continuity, differentiation, and integration.

## Core Topics

### Construction of ℝ
- Dedekind cuts
- Cauchy sequences
- Completeness axiom

### Topology of ℝ
- Open and closed sets
- Compactness
- Connectedness
- Heine-Borel theorem

### Sequences and Series
- Convergence criteria
- Bolzano-Weierstrass theorem
- Cauchy sequences
- Absolute vs conditional convergence

### Continuity
- Epsilon-delta definition
- Uniform continuity
- Intermediate value theorem
- Extreme value theorem

### Differentiation
- Rigorous definition
- Mean value theorem
- Taylor's theorem
- L'Hôpital's rule (rigorous)

### Integration
- Riemann integral
- Lebesgue integral
- Fundamental theorem (rigorous proof)
- Improper integrals

### Measure Theory
- Lebesgue measure
- Measurable functions
- Dominated convergence theorem
- Fubini's theorem
