# Analytical Chemistry

Methods for analyzing chemical substances.

## Overview

Analytical chemistry develops and applies methods to determine the composition and structure of matter.

## Core Topics

### Classical Methods
- Gravimetric analysis
- Volumetric analysis
- Titrations

### Spectroscopic Methods
- Atomic absorption (AA)
- Atomic emission (AES)
- UV-Visible spectroscopy
- Fluorescence spectroscopy
- X-ray methods

### Chromatography
- Gas chromatography (GC)
- High-performance liquid chromatography (HPLC)
- Thin-layer chromatography (TLC)
- Ion chromatography

### Mass Spectrometry
- Ionization methods
- Mass analyzers
- Interpretation
- Hyphenated techniques (GC-MS, LC-MS)

### Electrochemical Methods
- Potentiometry
- Voltammetry
- Conductometry
- Coulometry

### Sample Preparation
- Extraction
- Digestion
- Preconcentration

### Quality Assurance
- Calibration
- Standards
- Validation
- Uncertainty analysis

## Applications

- Environmental analysis
- Food safety
- Pharmaceutical quality control
- Forensic science
