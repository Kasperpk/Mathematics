# Neuroscience

The nervous system and brain.

## Overview

Neuroscience studies the structure and function of the nervous system, from molecules to cognition.

## Core Topics

### Cellular Neuroscience
- Neurons and glia
- Ion channels
- Action potentials
- Synaptic transmission

### Neuroanatomy
- Brain regions
- Spinal cord
- Cranial nerves
- White and gray matter

### Sensory Systems
- Vision
- Audition
- Somatosensation
- Chemical senses

### Motor Systems
- Spinal reflexes
- Voluntary movement
- Basal ganglia
- Cerebellum

### Cognitive Neuroscience
- Attention
- Memory
- Language
- Executive function

### Developmental Neuroscience
- Neural development
- Axon guidance
- Synaptogenesis
- Plasticity

### Computational Neuroscience
- Neural coding
- Network models
- Machine learning connections

### Disorders
- Neurodegenerative diseases
- Psychiatric disorders
- Neurological injuries
- Developmental disorders

## Techniques

- Electrophysiology
- Neuroimaging (fMRI, EEG, PET)
- Optogenetics
- Lesion studies

## Applications

- Brain-computer interfaces
- Neuroprosthetics
- Drug development
- Artificial intelligence
